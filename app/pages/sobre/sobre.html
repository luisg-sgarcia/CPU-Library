Sobre a CPU Library

Propósito do Projeto

A CPU Library é uma aplicação web desenvolvida com o objetivo de catalogar, gerenciar e exibir informações detalhadas sobre processadores. Ela permite aos usuários pesquisar, cadastrar novos modelos, editar informações existentes e excluir entradas, proporcionando uma ferramenta interativa para consulta e manutenção de dados de CPUs.

Contexto Acadêmico

Este projeto foi desenvolvido como parte da disciplina de "Desenvolvimento de Páginas Web com Framework e CSS". A disciplina é oferecida pela UTFPR Guarapuava (Universidade Tecnológica Federal do Paraná), Campus Guarapuava. O desenvolvimento da CPU Library visa aplicar conceitos e práticas de frameworks frontend e estilização CSS, bem como interação com APIs.
